Other Projects

As part of our training, we completed a project in pairs whose objective was to develop a program capable of automatically classifying news reports by topic. The main challenge was to optimize both processing speed and classification accuracy, using text analysis techniques. The program had to be able to identify the main topic of each report from among several categories.

The project was developed entirely in Java, which allowed us to deepen our mastery of object-oriented programming and data structures. We implemented text processing algorithms, such as word frequency calculations and similarity methods, to efficiently classify the news reports. Working in pairs also allowed us to develop our ability to collaborate and structure our code as a team. The report is available here.

Another project I've completed is a detailed installation guide for deploying a Debian 12 server on a virtual machine. The goal was to provide clear and accessible documentation for anyone wishing to install and configure Debian 12, following a series of precise steps. The guide includes all the basic configurations necessary to ensure an optimal and secure server installation, while also taking into account the various customization options possible during installation.

This project allowed me to strengthen my system administration skills, particularly in command line and Linux system management. I used the QEMU virtualization tool to simulate the installation and configure the environment. The most important aspect was to make the guide sufficiently detailed and easy to follow, explaining the various configuration options and providing solutions to common problems encountered during installation. The guide can be accessed here.
